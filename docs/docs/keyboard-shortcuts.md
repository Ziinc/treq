# Keyboard Shortcuts

Master Treq's keyboard shortcuts to work faster and more efficiently.

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl+K` | Open command palette |
| `Cmd/Ctrl+N` | New worktree |
| `Cmd/Ctrl+D` | Go to dashboard |
| `Cmd/Ctrl+R` | Refresh dashboard |
| `Cmd/Ctrl+,` | Open settings |
| `Cmd/Ctrl+W` | Close current tab |
| `Cmd/Ctrl+Q` | Quit Treq |

## Navigation

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl+1-9` | Switch to session 1-9 |
| `Cmd/Ctrl+[` | Previous session |
| `Cmd/Ctrl+]` | Next session |
| `Cmd/Ctrl+Tab` | Next tab |
| `Cmd/Ctrl+Shift+Tab` | Previous tab |

## Terminal

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl+C` | Copy selection (or interrupt) |
| `Cmd/Ctrl+V` | Paste |
| `Cmd/Ctrl+F` | Find in terminal |
| `Cmd/Ctrl+L` | Clear terminal |
| `Cmd/Ctrl+K` | Clear scrollback |
| `Cmd/Ctrl++` | Increase font size |
| `Cmd/Ctrl+-` | Decrease font size |
| `Cmd/Ctrl+0` | Reset font size |

## Git Operations

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl+Enter` | Commit (when message focused) |
| `Cmd/Ctrl+Shift+S` | Stage all |
| `Cmd/Ctrl+Shift+U` | Unstage all |
| `S` | Stage selected file |
| `U` | Unstage selected file |
| `C` | Focus commit message |

## Diff Viewer

| Shortcut | Action |
|----------|--------|
| `J`/`K` | Next/Previous hunk |
| `H` | Stage hunk |
| `Space` | Stage selected file |
| `Delete` | Discard changes |
| `Enter` | View file diff |
| `R` | Mark as reviewed |

## Code Review

| Shortcut | Action |
|----------|--------|
| `↓`/`↑` | Next/previous file |
| `C` | Add comment |
| `N`/`P` | Next/previous comment |
| `[`/`]` | Previous/next change |
| `A` | Approve changes |
| `X` | Request changes |

## Customizing Shortcuts

Settings → Keyboard Shortcuts → Customize

**Tips**:
1. Start with defaults
2. Customize conflicting shortcuts
3. Keep related actions similar
4. Document custom shortcuts for team

## Platform Differences

- **macOS**: `Cmd` is primary modifier
- **Windows/Linux**: `Ctrl` is primary modifier
- **Alternative**: `Option/Alt` for secondary actions
